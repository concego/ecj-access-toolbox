📊 Progress Bar

Gera barras de progresso acessíveis com ARIA correto. Três variantes: determinada, indeterminada e indicador de etapas.

Descreva O QUE está progredindo, não só "progresso"

🖼️ Image Alt Text

Gera o HTML correto para imagens com alt text, figcaption e imagens decorativas. #2 erro mais comum — 53.1% das imagens sem alt (WebAIM 2026).

Descreva o conteúdo e função — não "imagem de…".

Imagem decorativa recebe alt="" e role="presentation" — leitores de tela a ignoram.

Descreva a ação, não a aparência.
Colocada em um parágrafo oculto visualmente mas acessível por leitores de tela.
HTML puro — sem dependências

🔗 Accessible Link

Gera links com texto descritivo, aria-label quando necessário e atributos corretos para links externos. #3 erro mais comum — 46.3% dos links sem descrição (WebAIM 2026).

Deve fazer sentido fora de contexto — leitores de tela listam links isoladamente.
Substitui o texto visível para leitores de tela.
Boas práticas: informar formato e tamanho antes do download.
HTML puro — sem dependências

⏭️ Skip Link

Gera skip links e live regions — os dois padrões mais esquecidos que travam usuários de teclado e leitores de tela em todos os sites.

O elemento destino precisa ter tabindex="-1" para receber foco.
HTML puro — sem dependências
